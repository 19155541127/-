# 图形计算器 - Shape Calculator

这是一个用 Python 编写的图形计算器程序，是我**独立完成的第一个**项目，支持多种平面和立体图形的面积、周长及体积计算。

## 功能特点

### 平面图形计算

面积计算：矩形、三角形、圆形

周长计算：矩形、三角形、圆形

### 立体图形计算

体积计算：立方体、球体、圆锥体、圆柱体

## 技术特点

1.采用模块化函数设计，每个计算功能独立封装

2.多层菜单系统，用户交互清晰直观

3.异常处理机制，确保输入有效性

4.使用 Python 标准库 math 进行精确数学计算

## 使用方法

运行程序后，首先选择「平面图形」或「立体图形」

根据菜单提示，逐步选择要计算的图形类型

输入所需参数（如长、宽、半径、高等）

程序将自动计算并显示结果
